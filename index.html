WEB DESIGN POINT

3D 아이콘과 긴 타원, 푸른색의 메인 컬러로 통일감을 주었고
전체적으로 입체감있고 생동감있는 브랜드 특성을 살려 디자인하였습니다.

브랜드를 좀 더 친근하게 느끼면서도 가벼운 느낌이 아닌
다채로운 서비스라는 인식을 줄 수 있도록 구성하였습니다.

WEB DESIGN COMPOSITION

01

MAIN BANNER

플로팅되는 귀여운 느낌의 3D요소들을 넣고
강아지와 교감하는 느낌의 배너를 사용하여 친근감을 주었습니다.

02

RECOMMAND

다양한 프로그램을 사용자에게 추천해줌으로써 사용자가 빠르게
원하는 서비스를 쉽고 빠르게 찾을 수 있도록 구성하였습니다.

03

NEW SERVICE

새로운 서비스를 더 잘 알리기 위해
서비스 소개와 페이지 이동이 가능하도록 구성하였습니다.

04

EXPERT

전문성을 보이기 위해 전문가 이미지를 배치하고
신뢰감있는 블랙톤의 무드로 구성하였습니다.

05

PROGRAM

세분화 된 프로그램이 많기 때문에 큰 분류로 나누어
사용자가 전체 큰 분류를 한 눈에 볼 수 있도록 수정하였습니다.

06

REVIEW

견종을 확인할 수 있도록 썸네일에 이미지를 보여주고
블로그에서만 확인할 수 있는 블로그 리뷰를 볼 수 있게 구성하였습니다.

07

FOOTER

상단의 메인 배너와 통일감있는 디자인과 귀여운 요소들로
재밌게 표현될 수 있도록 구성하였습니다.

RESPONSIVE DESIGN

첫 웹 디자인 포트폴리오를 만들면서 브랜드의 특성을 고려하여
사용자에게 브랜드 가치와 메세지를 전달할 수 있도록 하고 싶었습니다.

실제로 포트폴리오를 디자인하고 또 구현해보면서 특색있는 요소 디자인을 할 때
콘텐츠의 배치와 구성이 주는 영향력과 컬러 팔레트, 폰트 선택 등이
사용자에게 얼마나 시각적으로 이해도를 높일 수 있는 지
그리고 그것을 코드로 어떻게 구현해야 할 지 생각해보는 뜻깊은 시간이었습니다.
